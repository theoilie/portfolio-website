Framework Summary

  • Worked with a more experienced dev to create a framework (Java) for other devs to make their own mini-games within Minecraft. This included game state logic (waiting/starting/playing/ending/spectating) and running multiple games concurrently.
  • Created utility functions for creating and saving/loading 3D arenas, forming teams, and customizing item collections.

TNT Run Summary

  • Used the framework to create a demo mini-game (Java) where players run around on falling blocks that eventually explode, and the last player to fall wins.
  • Partnered with a network (Arkham Network) to feature the game to their userbase of over 1 million unique players.

Plants Versus Zombies

  • Created my (Java) version of the mobile game "Plants Versus Zombies" in Minecraft, which received 7,000+ downloads on a gaming site (CurseForge). This involved implementing an optimization algorithm for assigning players to 1 of 2 teams with multiple rows to spawn in for balanced difficulty.
  • Created a ranking system to store information in a MySQL database and compare statistics such as win ratio.
