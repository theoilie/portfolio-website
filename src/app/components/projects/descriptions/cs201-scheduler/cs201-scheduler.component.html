Project Summary

  • Final project for a Java/web class. Allows student clubs and organizations to create calendars, share and edit them concurrently, and view conflicts with other organizations' calendars.

Personal Summary

  • Set up the Java Spring project with Maven, and provided documentation for the other students to install the dependencies (the team used both Windows and Mac, so we supported both)
  • Dockerized the project so that the MongoDB database shipped with it. This eliminated the need for new admins to install a database and made our demo process smoother
  • Created endpoints and functionality for user registration and login, as well as calendar creation, editing, and sharing
  • Used WebSockets (StompJS library) to implement concurrent calendar editing (like Google Docs but with calendar events instead of text editing)
  • Presented the project with my team to the class and professor for 10 minutes, and gave a live demo for 5 minutes
